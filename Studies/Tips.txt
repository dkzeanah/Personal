You can return objects using JSON syntax sugar too. I use it all the time.

fn(name, age)
{
  return {name: (name), age: (age)}
}

MsgBox, % fn("George", 15).name


fn(name, age)
{
  return [name, age]
}

MsgBox, % fn("George", 15)[1]
